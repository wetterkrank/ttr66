=== TTR66 Core ===
Contributors: wetterkrank

Core and UI components for TTR66.ru

== Description ==

Enables XML pricelist upload, adds a custom WP dashboard widget
Requires the TTR66 theme (Customify child)

== Installation ==

1. Upload the plugin to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
